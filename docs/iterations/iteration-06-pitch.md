### Page 1: 封面

### Page 2: 问题
**Agent出事后，企业拿不出证据。**

- 某银行的客服Agent泄露了客户信息→无法证明是Agent问题还是人为问题
- 某SaaS的Agent自动批准了异常操作→没有审计日志
- 某创业公司想签大客户→客户要求Agent合规证明→拿不出来

**现实：** Agent正在替代人工做决策，但决策过程没有记录、没有证据、没有审计能力。

### Page 3: 解决方案
**Lobster Blackbox — Agent行为记录仪**

- 录制Agent的每次决策（输入→推理→输出→工具调用）
- 自动脱敏+签名防篡改
- 一键生成审计报告（PDF+可验证签名）

```bash
npm install lobster-blackbox
```

## 3个致命问题的标准答案

**Q1: OpenAI自己做了怎么办？**
A: OpenAI只覆盖自家模型。全球30%的Agent用Claude/Gemini/本地模型，这些我们都能覆盖。而且OpenAI不做标准化审计报告，不做保险联动。

**Q2: 凭什么是你们？**
A: 我们不做认证（那是BSI的事），不做治理策略（那是Credo AI的事）。我们只做一件事：让Agent的每次决策都可追溯。这是技术活，不是关系活。

**Q3: 市场是不是太早了？**
A: 是的，2026是布道年。但正是因为早，才有机会建立标准。等市场成熟了再进来，就只能做价格竞争了。
